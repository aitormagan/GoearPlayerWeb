Descargar Playlist

Descargar una lista de canciones puede llegar a ser un proceso tedioso. Tienes dos opciones:

  1. Descargarte las canciones una a una
  2. Usar la versión de escritorio: Con la versión de escritorio puedes descargar una playlist de una forma rápida y cómoda. Para descargar esta lista lo único que tienes que hacer es abrir la aplicación e introducir playlist: y todas las canciones comenzarán a descargarse en el directorio que tú indiques sin que tengas que preocuparte de nada más.

Listas de favoritos

Acabas de añadir tu primera canción a la lista de favoritos. Antes de seguir añadiendo canciones vamos a explicarte un poco el funcionamiento de este sistema:

  1. Puedes añadir tantas canciones como quieras.
  2. Para borrar una canción de la lista de favoritos sólo tienes que pinchar en la canción que desees borrar y desmarcar la opción de favorita.
  3. Al hacer uso de este sistema estás aceptando el uso de almacenamiento local por parte de esta web. Se establecerá un item para guardar las canciones favoritas.
  4. Si utilizas navegación privada, cada vez que accedas a la página habrás perdido tu lista de canciones favoritas.
  5. Las listas de canciones son propias a cada navegador, es decir, tendrás una lista de canciones favoritas distinta en cada navegador que uses.
  6. No se puede compartir tu lista de canciones favoritas con amigos u otros dispositivos o navegadores.

Ayuda

Introducción:

GoearDownloader v 1.2 es una aplicación escrita en Java que te permite descargarte canciones del servicio Goear de manera muy sencilla a tu propio ordenador para que puedas sincronizar dichas canciones con otros dispositivos de audio tales como MP3 o teléfonos móviles o simplemente para disfrutar de las canciones en tu ordenador. El uso de tecnología Java para el funcionamiento del programa permite que pueda ser ejecutado en distintos Sistemas Operativos sin tener que hacer recompilaciones. Sin embargo, la desventaja de usar Java es que hay que instalar una máquina virtual que te permita ejecutar el archivo. Más adelante podrás encontrar un Anexo de cómo instalar Java en un Sistema Operativo Microsoft® Windows.

Debido a que el programa guarda una serie de datos para el funcionamiento en futuras sesiones se creará un archivo denominado “goeardownloader.conf”. Dicho archivo almacenará las preferencias que haya seleccionado el usuario. En caso de que el archivo sea modificado y su contenido no sea válido perderás tus preferencias ya que el programa generará automáticamente las preferencias por defecto para su Sistema Operativo. La primera vez que ejecutes el programa estas preferencias se crearán de forma automáticamente por lo que no tienes que preocuparte de crear este archivo en ningún caso.

Descargar Canciones:

A continuación se ofrece una visión de la pantalla que verás cuando habrá el programa para proceder a la descarga de canciones:

Para proceder a descargar una canción con el programa lo primero que debes hacer es entrar en http://www.goear.com. Allí debes buscar la canción que quieres descargar y seleccionar una de las múltiples opciones que te ofrece Goear. Una vez te encuentres escuchando la canción deseada introduce la URL que sale en el explorador en el cuadro donde pone “Introduzca aquí la dirección de la canción”. Puedes introducir la dirección de forma manual escribiéndola en el cuadro, pero el programa está preparado para que cuando haya una dirección de Goear en el portapapeles, al hacer clic en el cuadro, la dirección se escriba automáticamente. Si por algún motivo esto no funcionase y no quieres escribir la dirección en el cuadro de forma manual, hemos creado un menú llamado “Editar” donde encontrarás las opciones típicas de un editor de texto (Cortar, Copiar, Pegar y Seleccionar Todo). Cuando en el cuadro de texto se encuentre la dirección de Goear sólo tienes que pulsar “Enter” en el teclado o el botón “Comenzar Descarga”. El programa comenzará a analizar la dirección introducida. Se producirá una conexión a los servidores de Goear que permitirá determinar si la dirección introducida es correcta y además se podrá determinar cuál es la dirección del archivo MP3.

En caso de que la dirección sea incorrecta (o que no se disponga de conexión a Internet) se mostrará un mensaje de error informando del error que se haya producido. En caso contrario, y según las preferencias del usuario, se mostrará un mensaje para que seleccione la carpeta donde se descargará la canción (este cuadro puede no llegar a mostrarse si el usuario así lo decide seleccionando que las canciones se descarguen automáticamente en una carpeta directamente). Cuando se haya seleccionado la carpeta de descarga se procederá a la descarga del archivo. En la parte inferior del programa podrás ver cuál es el nombre y artista de la canción, el porcentaje de la descarga, la cantidad de información que ya ha sido descargada, el tiempo estimado que resta para completar la descarga y la velocidad media de descarga.

GoearDownloader ofrece además multidescarga que te permite descargar varias canciones simultáneamente. Cuando una canción esté siendo bajada podrás iniciar la descarga de otra canción (o una lista de canciones) introduciendo su dirección en la barra de descargas.

Introducir Identificador: Es posible que en algunas situaciones GoearDownloader no sea capaz de detectar el identificador de la canción que deseas descargar. El identificador de una canción de Goear es un código de siete caracteres alfanuméricos. Es muy fácil de identificar y es que en una dirección tipo: http://www.goear.com/listen/3f253c0/i-gotta-feeling-black-eyed-peas, el código es el conjunto de caracteres presentes entre /listen/ y la siguiente /. En este caso el identificador sería 3f253co. Con el identificador, puedes empezar la descarga de una canción de Goear, introduciendo en el campo de descargas lo siguiente: song:goearID, donde goearID será el identificador de la canción de Goear. Para el ejemplo anterior tendríamos que introducir lo siguiente: song:3f253co.

Descargar Listas de Canciones:

A partir de la versión 1.2 de GoearDownloader es posible descargar listas de canciones. De la misma forma que con las descargas individuales, para descargar una lista de canciones debes introducir la dirección en el cuadro de descargas. Aparecerá un nuevo panel donde se informará en todo momento el estado de la descarga de la lista. El programa analizará todas las canciones que componen la lista y procederá a descargarlas una a una. En este caso, cuando se arranque la descarga de una lista, no se podrán descargar más archivos hasta que no hayan arrancado todas las descargas de las canciones que componen la lista.

Introducir Identificador: Es posible que en algunas situaciones GoearDownloader no sea capaz de detectar el identificador de la lista que deseas descargar. El identificador de una lista de Goear es un código de siete caracteres alfanuméricos. Es muy fácil de identificar y es que en una dirección tipo: http://www.goear.com/playlist.php?v=1f27838, el código es el conjunto de caracteres presentes después de ?v=. En este caso el identificador sería 1f27838. Con el identificador, puedes empezar la descarga de una lista de Goear, introduciendo en el campo de descargas lo siguiente: playlist:goearID, donde goearID será el identificador de la lista de Goear. Para el ejemplo anterior tendríamos que introducir lo siguiente: playlist:1f27838.

Herramientas:

El programa dispone de una serie de herramientas que pueden hacerle más placido el uso de este programa. Al hacer clic en “Herramientas” se desplegará el siguiente menú:

La opción “Ventana Siempre Visible” le permitirá que la ventana siempre este visible aunque esté trabajando con otros programas. El programa permanecerá visible siempre que no minimice la ventana o que no esté en mitad de una descarga.

La opción “Posición Automática” permite al programa colocarse en medio de la pantalla cada vez que se produce un evento tal como comenzar o terminar una descarga de una canción.

La opción "Notificar Error Dirección XML" abrirá una incidencia en la página indicando que la dirección de búsqueda de los XML disponible en el servidor no es correcta y por tanto no se pueden realizar descargas con normalidad.

La opción "Desactivar Descarga Automática" sólo estará activa mientras este activa la descarga automática. Si seleccionas dicha opción, aunque la descarga automática este activada, se te preguntará la dirección donde descargar la canción hasta que reinicies el programa o desactives dicha opción. Con esto puedes editar el nombre de la canción por si existe algún error en alguna descarga específica... Si activas esta opción la dirección de descarga no se actualizará para futuras sesiones y se mostrará como carpeta de descarga la carpeta donde se descargo la última canción (como si se hubiese activado el modo "Conservar Dirección de Descarga durante la misma sesión"). Si quieres que se almacene la dirección, puedes usar cualquiera del resto de "Opciones de Descarga" disponible en el menú de configuración del programa (a continuación...).

Al hacer clic en “Opciones” se abrirá el siguiente diálogo:

Como puede observar, la ventana se encuentra divida en tres grandes secciones: Opciones de Descarga, Solucionar Problemas con las Descargas y Actualizaciones. Estas secciones se detallarán a continuación.

Opciones de Descarga: Esta opción se refiere a las opciones de descarga del programa que le permitirán elegir por ejemplo cuál es la dirección donde siempre deberán descargando las canciones:

  1. El cuadro de texto “Dirección Predeterminada de Descarga” contiene la dirección que siempre mostrará el seleccionador de archivos cuando descargues una canción con el programa. Puedes cambiar esta carpeta haciendo clic en el botón “…”.
  2. Si selecciona “Descargar Siempre en Dirección Predeterminada Descarga sin preguntar” el programa no mostrará ningún seleccionador de archivos para que indique donde ha de guardarse la canción que descargue sino que se descargará en la “Dirección Predeterminada de Descarga” automáticamente con el nombre “Artista – Canción.mp3”, sustituyendo “Artista” por el artista de la canción y “Canción” por el nombre de la canción que está descargando.
  3. Si selecciona “Actualizar Automáticamente Dirección Predeterminada de Descarga” y cuando descargas una canción cambias la carpeta dónde se descarga dicha canción, la nueva carpeta se convertirá automáticamente en la nueva “Dirección Predeterminada de Descarga”.
  4. La opción “Conservar Dirección de Descarga durante la misma sesión” es muy similar a la anterior, pero la carpeta sólo se almacena durante la sesión en funcionamiento. Al iniciar de nuevo el programa la carpeta que mostrará la primera canción descargada será la “Dirección Predeterminada de Descarga”.

Solucionar Problemas de Descarga: Debido a los diversos cambios en Goear se ha creado este menú que permite al programa buscar soluciones a posibles cambios en Goear conectándose al servidor del programa. Si observas que las canciones que se descargan no son las deseadas haga clic en el botón “Auto Solucionar”. El programa se conectará al servidor y buscará la nueva dirección de búsqueda de los XML. En caso de no encontrar ninguna dirección nueva o que no se pueda conectar con el servidor se permitirá editar la dirección de forma manual, aunque se avisará de que esa acción puede dejar el programa inutilizable. En caso de encontrar una dirección de búsqueda de los XML nueva se pondrá en el cuadro de texto y se instará al usuario a guardar las preferencias para descargar las canciones de forma correcta. Además se provee una opción que hará que se busquen soluciones a problemas de descarga siempre que se inicie el programa. Si selecciona esta opción no tienes que preocuparse por posibles errores en las descargas, ya que siempre al iniciar se buscará la dirección de XML más reciente. Cuando se estén buscando Soluciones a problemas de descarga de forma automática al iniciar verás un mensaje informando de ello en la barra de estado:

Actualizaciones: Esta opción permitirá al programa buscar actualizaciones de forma automática al programa al iniciarse. Cuando se estén buscando actualizaciones de forma automática al iniciar verás un mensaje informando de ello en la barra de estado:

En caso de encontrase actualizaciones se pedirá confirmación al usuario para poder actualizar.

Si el usuario acepta la actualización se buscará el archivo “goeardownloader.jar” dentro del mismo directorio dónde se está ejecutando el programa. Si el archivo no es encontrado se procederá a instar al usuario a actualizar el programa de forma manual. Si por el contrario el archivo existe el programa será actualizado siempre que no falle la conexión a Internet.

Una vez se haya actualizado el programa se ejecutará la nueva versión del programa. En algunos casos es posible que tras la actualización, la nueva versión no se cargue de forma automática. En dichos casos lo único que el usuario debe hacer es volver a cargar el programa.

NOTA: el usuario puede buscar actualizaciones de forma manual haciendo clic en “Ayuda” > “Buscar Actualizaciones”. El proceso es el mismo que una actualización automática. (esta opción esta disponible sólo cuando no hay descargas activas)

NOTA 2: En versiones superiores a la 100623, en estos diálogos puede mostrarse un botón denominado "Cancelar" que sirve para cancelar el proceso en curso. Sin embargo, de momento, dicho botón no es del todo funcional ya que en algunas situaciones el proceso en curso no puede detenerse por una llamada bloqueante al sistema. Próximamente intentaremos solucionar este error.

Mandar Informes de Error:

Bajo diversas circunstancias es posible que se produzcan errores durante la descarga o la ejecución del programa. Para poder solucionar estos problemas es muy importante la colaboración de los usuarios del programa. En ningún caso se enviará información a nuestros servidores sin tu confirmación expresa para cada error que se produzca. Cuando se produzca un error que no debería producirse el usuario verá un cuadro similar al siguiente:

Para poder mandar información del error al servidor del programa es necesario que pulses el botón de “Aceptar”. En caso de disponer conexión a Internet y no producirse ningún error la información llegará de forma correcta al servidor. Siempre que se pulse “Aceptar” aunque se produzca un error se agradecerá la colaboración del usuario. En caso de no aceptar el envío de la información se recordará al usuario que sin su colaboración la mejora del programa no es posible.

Si decidiste enviar la información, el programa te ofrecerá un identificador de error mediante el cual podrás seguir el estado de tu incidencia (siempre que no haya un problema en el servidor). Para ver el estado de la incidencia haz clic en "Informes de Error", una página que encontrarás en el menú. Allí introduce tu código y podrás ver el estado de tu incidencia así como comentarios del administrador a dicha incidencia.

NOTA: En ningún caso se almacenará en el servidor ninguna información que pueda identificar quien mandó el reporte.

Anexo - Instalando Java:

Como ya hemos dicho anteriormente, para poder ejecutar el programa es necesario tener instalado en tu ordenador la máquina virtual de Java (se recomienda la diseñada por Sun).

Lo primero que tienes que hacer para instalar la máquina virtual es bajártela a tu sistema local. Para ello debes entrar en la página web oficial de Java. Una vez allí deberás pinchar el botón denominado "Descargas" que encontrarás en la parte superior de la página. La página reconocerá automáticamente tu S.O. y en función a ello te ofrecerá la versión de Java correspondiente. Pincha entonces en "Descarga gratuita de Java" y espera a que se baje el archivo. Una vez que el archivo se haya bajado completamente debes instalarlo en tu ordenador como cualquier programa. ¡¡¡Tras finalizar ya podrás usar GoearDownloader perfectamente!!!

Acerca de GoearDownloader

GoearDownlaoder es un servicio web que te permite buscar y descargar canciones de manera sencilla del servicio Goear. Además, existen dos aplicaciones, una para equipos y otra para smartphones Android, que te permitirán buscar y descargar la música de una manera mucho más rápida.
